# Triage

## Feature Requests and Usage Questions

```
If you license the professional version of this product, full support is available:
https://polangeles.com/duty_free

Otherwise, please raise feature requests and issues on [GitHub][1] and ask usage
questions on [StackOverflow][2].  Bear in mind that Polangeles, at its discretion,
may choose to fulfil some feature requests only in DutyFree Professional.  The
company's aim over time is to create a fully-fledged suite of ETL and reporting
solutions that easily interoperate with one another.

When filing a bug report, please use the [bug report template][3].

[1]: https://github.com/lorint/duty_free/issues
[2]: https://stackoverflow.com/tags/duty-free-gem
[3]: https://github.com/lorint/duty_free/blob/master/doc/bug_report_template.rb
```
